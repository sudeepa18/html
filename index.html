Java Short Notes

Introduction

Java is a high-level,object-oriented,and platform independent programming language. It was developed by James Gosling in 1995.

Features of Java

  • Simple- Easy to learn & write
  • Object-Oriented- Everything is treated as an object
  • Platform Independent-"Write once , run anywhere"
  • Secure- Provides built-in security

Data Types

Type Description Example
int Stores integers int num = 10;
double Stores floating point numbers double pi = 3.14;
boolean Stores true or false values
